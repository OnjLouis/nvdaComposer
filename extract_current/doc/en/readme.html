NVDA Composer 1.3

NVDA Composer is a keyboard-first, screen-reader-friendly music sketchpad inside NVDA. You enter notes (or rests) on a timeline, select and edit them, and play back your project. Projects can be saved to a plain-text format (.txt), imported from MIDI (.mid/.midi) or Nokia ringing tones (.rng), and exported as MIDI, WAV, or Nokia .rng.

Download: All Andre add-ons are available from https://onj.me/nvda

Changelog

Version 1.3

  • New: Velocity / volume support (adjust with comma/period, Shift+comma/period for larger steps, and press V to type a velocity directly).
  • Improved: Share clips now start with #NVDAComposer, use a more compact format, and clipboard import is more forgiving if extra text is copied above/below the clip.

Version 1.2

  • Custom durations and step length: press ` to enter an exact custom step (for new notes/rests), or Ctrl+` to set an exact duration for the current note or selection.
  • Track properties: F2 lets you set an optional project title and notes/comments (saved with the project).
  • Share-friendly clipboard: Ctrl+Shift+C copies a clip that can include title/tempo/notes as comment lines.
  • Paste with metadata: Ctrl+Shift+V pastes and offers to import the clip’s title/tempo/notes into the current project.
  • More tolerant clip import: Clipboard imports ignore harmless extra text (blank lines, comments, and unknown lines), making social sharing more robust.

Also included (from the 1.0 release cycle)

  • Save As: Ctrl+Shift+S (and F12) saves a copy under a new name.
  • Nokia ringing tones: import and export .rng via Ctrl+O and Ctrl+E.
  • Markers: Ctrl+M toggles a marker; -/= jump between markers.
  • Go to: Ctrl+G jumps to a specific position on the timeline.
  • Reverse (Easter egg): Shift+Backspace flips the timeline end-to-start.
  • Help inside the add-on: F1 opens this documentation; Shift+F1 opens the quick tutorial.
  • Open demos quickly: Ctrl+Shift+O opens the file dialog directly in the demos folder.
  • Playback follow modes: Ctrl+F toggles Follow playback; Ctrl+Shift+F toggles Snap on pause.

Project formats

  • NVDA Composer project (.txt) – native save format.
  • MIDI (.mid, .midi) – import/export.
  • Nokia ringing tone (.rng) – import and export (monophonic; patterns/loops are flattened on import).
  • WAV (.wav) – export. (Includes a Nokia-style tone option.)

Open the Composer

  • NVDA+Alt+N: Open NVDA Composer.

Entering notes

NVDA Composer is designed for fast, screen-reader-friendly note entry. The basic loop is: choose a step length → move the cursor → enter notes/rests → play back.

Step length (new notes and rests)

  • 1–6: set step length (whole, half, quarter, eighth, sixteenth, thirty-second).
  • 0: toggle dotted / undotted step.
  • ` (grave; on some keyboards this is ¬): type a custom step length.

Input pitch and inserting

  • Left / Right: move the timeline cursor by one step.
  • Press a note key to insert a note at the cursor (then the cursor advances by one step).
  • Space: insert a rest.
  • Up / Down: shift the input pitch by 1 semitone (affects new notes).
  • Alt+Up / Alt+Down: shift the input pitch by 1 octave.

Grid vs QWERTY layouts (Ctrl+K)

Ctrl+K switches the note-entry keyboard:

  • Grid layout (default): a compact 3×3 “pad” on J K L / U I O / 7 8 9.
  • QWERTY layout: a longer “piano-ish” row on A W S E D F T G Y H U J K.

The layout only changes which keys insert notes; everything else (step length, playback, selection, etc.) stays the same.

Chromatic vs Scale mode

  • Ctrl+T: toggle Chromatic (semitones) vs Scale mode (scale degrees).
  • Alt+Left / Alt+Right: change the current scale (used in Scale mode).

Key maps (reference)

In Chromatic mode, the keys are semitone offsets from the current input pitch. In Scale mode, they are scale degrees (0 = root).

LayoutKeys that insert notesChromaticScale
Grid J K L U I O 7 8 9 Offsets 0–8 Degrees 0–8
QWERTY A W S E D F T G Y H U J K Offsets 0–12 A S D F G H J K = degrees 0–7

Clipboard

KeyAction
Ctrl+CCopy selected events (or the current event if nothing is selected) to the system clipboard.
Ctrl+VPaste clipboard events at the cursor position.
Ctrl+Shift+CCopy a share clip (includes optional title/tempo/notes as comment lines).
Ctrl+Shift+VPaste, and offer to import the clip’s title/tempo/notes into the current project.

Because the system clipboard is used, you can copy from one Composer window and paste into another. Clips may also include harmless comment lines (starting with # or ;) which are ignored on import.

Track properties (title and notes)

KeyAction
F2Edit the project title and notes/comments. The title is used for the project announcement when available; otherwise the filename is announced.

Share clips

  • Use Ctrl+Shift+C to copy a share clip. If you have a selection, only that range is copied; otherwise the whole project is copied.
  • Use Ctrl+Shift+V to paste a share clip. If it contains # tempo:, # title:, or # notes: comment lines, you can choose whether to import them.
  • Tip for social media: paste the clip inside a code block (triple backticks) to preserve line breaks.
  • Clipboard import tolerates blank lines, comment lines (#, ;, //), unknown lines, and even clips where newlines were turned into literal \n.

Velocity / volume

KeyAction
, / .Decrease / increase velocity (volume) for new notes. If you have a selection, changes velocity for the selected notes instead.
Shift+, / Shift+.Decrease / increase velocity in larger steps.
VType a velocity value (1–127). Applies to selected notes if you have a selection; otherwise sets the input velocity for new notes.

Velocity is written to MIDI exports, WAV renders, and Nokia .rng volume instructions.

Step length for new notes/rests

KeyAction
1 … 6Set step for new notes/rests to Whole/Half/Quarter/Eighth/Sixteenth/Thirty-second.

Custom duration input

KeyAction
`Enter a custom step length for new notes/rests.
Ctrl+`Set an exact duration for the current note (or the selection).

Accepted inputs: ticks (e.g. 333 or 333t), beats as quarter-notes (e.g. 1.5b), beats with explicit PPQ (e.g. 1.5b480), or fractions of a whole note (e.g. 3/16). On UK layouts, the same key may be labelled ¬.

Formats

MIDI import/export

  • Open/import: Ctrl+O supports multi-track MIDI. All tracks are merged and made monophonic.
  • Export: Ctrl+E writes MIDI (and tempo changes if present).

Nokia ringing tone (.rng) import

  • Open/import: Ctrl+O can import Nokia/Smart Messaging ringing tones (.rng).
  • Ringing tones are monophonic; patterns/loops are flattened into a single note/rest timeline.

Nokia ringing tone (.rng) export

  • Export: Ctrl+E → choose Nokia (.rng) to export a ringtone suitable for classic Smart Messaging phones.
  • Export is monophonic, and durations are quantized to what the format supports.

WAV export

  • Ctrl+E → choose WAV to render an audio file of your project.
  • The export dialog includes a standard tone render and a “Nokia-style” tone render.
